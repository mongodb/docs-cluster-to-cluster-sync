==========
``resume``
==========

.. include:: /includes/preview-warning

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Description
-----------

Resumes a paused synchronization session based on data stored on the
destination cluster.

Requirement
-----------

To use the ``resume`` endpoint, the {+c2c-product-name+} must be in the
``PAUSED`` state.

Request
-------

.. code-block:: http

   POST /api/v1/resume

Behavior

``RUNNING`` state.
